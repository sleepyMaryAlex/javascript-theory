# ?Sets and ranges

–ù–µ—Å–∫–æ–ª—å–∫–æ —Å–∏–º–≤–æ–ª–æ–≤ –∏–ª–∏ —Å–∏–º–≤–æ–ª—å–Ω—ã—Ö –∫–ª–∞—Å—Å–æ–≤ –≤ –∫–≤–∞–¥—Ä–∞—Ç–Ω—ã—Ö —Å–∫–æ–±–∫–∞—Ö `[...]` –æ–∑–Ω–∞—á–∞—é—Ç ¬´–∏—Å–∫–∞—Ç—å –ª—é–±–æ–π —Å–∏–º–≤–æ–ª –∏–∑ –∑–∞–¥–∞–Ω–Ω—ã—Ö¬ª.

#### –ù–∞–±–æ—Ä—ã

–î–ª—è –ø—Ä–∏–º–µ—Ä–∞, `[eao]` –æ–∑–Ω–∞—á–∞–µ—Ç –ª—é–±–æ–π –∏–∑ 3-—Ö —Å–∏–º–≤–æ–ª–æ–≤: `'a'`, `'e'` –∏–ª–∏ `'o'`.

–ù–∞–±–æ—Ä—ã –º–æ–≥—É—Ç –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å—Å—è –≤ —Ä–µ–≥—É–ª—è—Ä–Ω—ã—Ö –≤—ã—Ä–∞–∂–µ–Ω–∏—è—Ö –≤–º–µ—Å—Ç–µ —Å –æ–±—ã—á–Ω—ã–º–∏ —Å–∏–º–≤–æ–ª–∞–º–∏, –Ω–∞–ø—Ä–∏–º–µ—Ä:

~~~
console.log("Cat mat".match(/[cm]at/gi)); // ['Cat', 'mat']
~~~

#### –î–∏–∞–ø–∞–∑–æ–Ω—ã

–ï—â—ë –∫–≤–∞–¥—Ä–∞—Ç–Ω—ã–µ —Å–∫–æ–±–∫–∏ –º–æ–≥—É—Ç —Å–æ–¥–µ—Ä–∂–∞—Ç—å –¥–∏–∞–ø–∞–∑–æ–Ω—ã —Å–∏–º–≤–æ–ª–æ–≤.

–ö –ø—Ä–∏–º–µ—Ä—É, `[a-z]` —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É–µ—Ç —Å–∏–º–≤–æ–ª—É –≤ –¥–∏–∞–ø–∞–∑–æ–Ω–µ –æ—Ç `a` –¥–æ `z`, –∏–ª–∏ `[0-5]` ‚Äì —Ü–∏—Ñ—Ä–∞ –æ—Ç `0` –¥–æ `5`.

–í –ø—Ä–∏–≤–µ–¥—ë–Ω–Ω–æ–º –Ω–∏–∂–µ –ø—Ä–∏–º–µ—Ä–µ –º—ã –∏—â–µ–º `"x"`, –∑–∞ –∫–æ—Ç–æ—Ä—ã–º —Å–ª–µ–¥—É—é—Ç –¥–≤–µ —Ü–∏—Ñ—Ä—ã –∏–ª–∏ –±—É–∫–≤—ã –æ—Ç `A` –¥–æ `F`:

~~~
console.log("Exception 0xAF".match(/x[0-9A-F][0-9A-F]/g)); // ['xAF']
~~~

–ó–¥–µ—Å—å –≤ `[0-9A-F]` —Å—Ä–∞–∑—É –¥–≤–∞ –¥–∏–∞–ø–∞–∑–æ–Ω–∞: –∏—â–µ—Ç—Å—è —Å–∏–º–≤–æ–ª, –∫–æ—Ç–æ—Ä—ã–π –ª–∏–±–æ —Ü–∏—Ñ—Ä–∞ –æ—Ç `0` –¥–æ `9`, –ª–∏–±–æ –±—É–∫–≤–∞ –æ—Ç `A` –¥–æ `F`.

–¢–∞–∫–∂–µ –º—ã –º–æ–∂–µ–º –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å —Å–∏–º–≤–æ–ª—å–Ω—ã–µ –∫–ª–∞—Å—Å—ã –≤–Ω—É—Ç—Ä–∏ `[...]`.

–ù–∞–ø—Ä–∏–º–µ—Ä:

* `\d` ‚Äì —Ç–æ –∂–µ —Å–∞–º–æ–µ, —á—Ç–æ –∏ `[0-9]`,
* `\w` ‚Äì —Ç–æ –∂–µ —Å–∞–º–æ–µ, —á—Ç–æ –∏ `[a-zA-Z0-9_]`,
* `\s` ‚Äì —Ç–æ –∂–µ —Å–∞–º–æ–µ, —á—Ç–æ –∏ `[\t\n\v\f\r ]`, –ø–ª—é—Å –Ω–µ—Å–∫–æ–ª—å–∫–æ —Ä–µ–¥–∫–∏—Ö –ø—Ä–æ–±–µ–ª—å–Ω—ã—Ö —Å–∏–º–≤–æ–ª–æ–≤ –Æ–Ω–∏–∫–æ–¥–∞.

#### –ü—Ä–∏–º–µ—Ä: –º–Ω–æ–≥–æ—è–∑—ã—á–Ω—ã–π –∞–Ω–∞–ª–æ–≥ `\w`

–¢–∞–∫ –∫–∞–∫ —Å–∏–º–≤–æ–ª—å–Ω—ã–π –∫–ª–∞—Å—Å `\w` —è–≤–ª—è–µ—Ç—Å—è –≤—Å–µ–≥–æ –ª–∏—à—å —Å–æ–∫—Ä–∞—â–µ–Ω–∏–µ–º –¥–ª—è `[a-zA-Z0-9_]`, –æ–Ω –Ω–µ –Ω–∞–π–¥—ë—Ç –∫–∏—Ç–∞–π—Å–∫–∏–µ –∏–µ—Ä–æ–≥–ª–∏—Ñ—ã, –∫–∏—Ä–∏–ª–ª–∏—á–µ—Å–∫–∏–µ –±—É–∫–≤—ã –∏ —Ç.–ø.

–î–∞–≤–∞–π—Ç–µ —Å–¥–µ–ª–∞–µ–º –±–æ–ª–µ–µ —É–Ω–∏–≤–µ—Ä—Å–∞–ª—å–Ω—ã–π —à–∞–±–ª–æ–Ω, –∫–æ—Ç–æ—Ä—ã–π –∏—â–µ—Ç —Å–∏–º–≤–æ–ª—ã, –∏—Å–ø–æ–ª—å–∑—É–µ–º—ã–µ –≤ —Å–ª–æ–≤–∞—Ö, –¥–ª—è –ª—é–±–æ–≥–æ —è–∑—ã–∫–∞. –≠—Ç–æ –æ—á–µ–Ω—å –ª–µ–≥–∫–æ —Å –Æ–Ω–∏–∫–æ–¥-—Å–≤–æ–π—Å—Ç–≤–∞–º–∏: `[\p{Alpha}\p{M}\p{Nd}\p{Pc}\p{Join_C}]`.

~~~

// –Ω–∞–π–¥–µ–Ω—ã –≤—Å–µ –±—É–∫–≤—ã –∏ —Ü–∏—Ñ—Ä—ã
console.log(str.match(regexp)); // ['H', 'i', '‰Ω†', 'Â•Ω', '1', '2']
~~~

#### –ò—Å–∫–ª—é—á–∞—é—â–∏–µ –¥–∏–∞–ø–∞–∑–æ–Ω—ã

–ü–æ–º–∏–º–æ –æ–±—ã—á–Ω—ã—Ö –¥–∏–∞–ø–∞–∑–æ–Ω–æ–≤, –µ—Å—Ç—å ¬´–∏—Å–∫–ª—é—á–∞—é—â–∏–µ¬ª –¥–∏–∞–ø–∞–∑–æ–Ω—ã, –∫–æ—Ç–æ—Ä—ã–µ –≤—ã–≥–ª—è–¥—è—Ç –∫–∞–∫ `[^...]`.

–û–Ω–∏ –æ–±–æ–∑–Ω–∞—á–∞—é—Ç—Å—è —Å–∏–º–≤–æ–ª–æ–º –∫–∞—Ä–µ—Ç–∫–∏ `^` –≤ –Ω–∞—á–∞–ª–µ –¥–∏–∞–ø–∞–∑–æ–Ω–∞ –∏ —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É—é—Ç –ª—é–±–æ–º—É —Å–∏–º–≤–æ–ª—É –∑–∞ –∏—Å–∫–ª—é—á–µ–Ω–∏–µ–º –∑–∞–¥–∞–Ω–Ω—ã—Ö.

~~~

~~~

#### –≠–∫—Ä–∞–Ω–∏—Ä–æ–≤–∞–Ω–∏–µ –≤–Ω—É—Ç—Ä–∏ `[...]`

–í –∫–≤–∞–¥—Ä–∞—Ç–Ω—ã—Ö —Å–∫–æ–±–∫–∞—Ö –±–æ–ª—å—à–∏–Ω—Å—Ç–≤–æ —Å–ø–µ—Ü–∏–∞–ª—å–Ω—ã—Ö —Å–∏–º–≤–æ–ª–æ–≤ –º–æ–∂–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –±–µ–∑ —ç–∫—Ä–∞–Ω–∏—Ä–æ–≤–∞–Ω–∏—è:
